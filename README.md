## heading
Headings are like html  
\# = \<h1>\</h1>  
\## = \<h2>\</h2>  
\### = \<h3>\</h3>  
\#### = \<h4>\</h4>  
\##### = \<h5>\</h5>  
\###### = \<h6>\</h6>  
## Bold
\*\*text\*\*   
\_\_text\_\_    
**this is bold text**  
they are just \<b>\</b> in html
## Italic
\*text\*  
\_text\_  
*this is italic text*
## Strikethrough
\~\~text\~\~  
~~this is strikethrough text~~
## soft break
this is hard break

in hard break we have one line without anything  
this is soft break  
we use 2 space after line to do soft break and continue in the next line
